How Voter ID Cards Work

Click to expand / collapse
📋 Before Election Day
  1. Add all voters to the Voters Register with their photo and Voter ID
  2. Print ID cards from this page — 6 or 8 per A4 sheet
  3. Cut cards along the borders with scissors
  4. Laminate the cards if possible for durability
  5. Distribute one card to each registered voter before election day
🗳️ On Election Day
  1. Each voter presents their ID card to the polling officer
  2. Polling officer checks the Voter ID number against the Voters Register
  3. Voter's name is ticked off in the register
  4. Voter receives their ballot paper and proceeds to vote
  5. ID card may be retained by the officer or returned to the voter
💡 Tips
  • Use the photo toggle to include voter photos for easy identification
  • Print by class or group to distribute class-by-class
  • Each Voter ID is unique — no two voters share an ID
  • Lost card? Reprint from this page using the voter's name filter
  • Keep printed cards confidential until distribution day
